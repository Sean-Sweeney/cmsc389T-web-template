Projects

Summer Camp Application

I developed a full-stack web application designed for users to submit and manage applications for a summer camp. This platform featured secure user authentication through OAuth, facilitating logins via Google and Facebook. Additionally, I integrated the Twilio API to dispatch real-time SMS notifications, updating users about the status of their applications. To ensure fluid client-server interaction, I crafted endpoints and fine-tuned database functions using Node.js.

NFL Salary Insights

I conducted a comprehensive analysis of NFL salaries in relation to player performance. By quantifying the value of each position, I delved deep into specific metrics such as passing, rushing, and defense. This analysis provided vital insights into how team schemes and strategies influence player valuation. Using these insights, I formulated predictive models to determine a player's worth, offering a strategic guide for future salary allocations.

Air Pollution Alert

I harnesses the power of the cloud, IoT, and geolocation to offer users instantaneous updates on local air quality, pulling from authoritative sources such as the EPA and satellite imagery. Built on a robust backend with Python and Node.js, the mobile application, crafted with React, enables users to visualize pollution hotspots, ensuring both health safety and environmental awareness.
